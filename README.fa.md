# سان‌لاین‌نت (SunLionet)

> شبکه‌ای آزاد برای جهانی آزاد

**سان‌لاین‌نت (SunLionet)** نام جدید پروژه‌ای است که پیش‌تر با نام ShadowNet شناخته می‌شد. این سیستم یک پلتفرم آفلاین‌محور و مبتنی بر بسته (bundle) برای مقابله با سانسور و DPI در محیط‌های پرریسک است.

در حال حاضر، نام مخزن هنوز `shadownet-agent` است و برخی از نام باینری‌ها و مسیرهای داخلی ممکن است همچنان از `shadownet-*` استفاده کنند تا زمانی که فرآیند مهاجرت کامل شود.

این پروژه شامل اجزای زیر است:

- **SunLionet Inside (نسخه داخل)**: روی دستگاه کاربر اجرا می‌شود و با تشخیص اختلال، تنظیمات را به‌صورت محلی تغییر می‌دهد.
- **SunLionet Outside (نسخه خارج)**: برای تولید و ارسال بسته‌های امن و امضاشده توسط پشتیبان‌ها استفاده می‌شود.
- **SunLionet App**: نسخه اندروید برای استفاده آسان‌تر از Inside.
- **SunLionet Dashboard**: رابط وب برای مشاهده وضعیت سیستم به‌صورت محلی.

آدرس فعلی مستندات:
https://kaveh8866.github.io/shadownet-agent/

پس از تغییر نام مخزن، به مسیر جدید منتقل خواهد شد.

## هدف پروژه

سان‌لاین‌نت برای فراهم کردن موارد زیر توسعه داده می‌شود:

- ارتباطات امن
- دسترسی به اطلاعات بدون سانسور
- حفظ حریم خصوصی
- قابلیت استفاده برای کاربران فارسی‌زبان

این پروژه به‌طور ویژه با در نظر گرفتن شرایط محدودکننده دیجیتال، از جمله ایران، طراحی شده است.

## بیلد

در حال حاضر نام باینری‌ها تغییر نکرده است:

```bash
go build -tags inside -o bin/shadownet-inside ./cmd/inside/
```

```bash
go build -tags outside -o bin/shadownet-outside ./cmd/outside/
```
