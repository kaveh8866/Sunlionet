# SunLionet Safety Guide

## Important Limits

- SunLionet does **not** guarantee anonymity.
- SunLionet does **not** provide legal protection.
- SunLionet does **not** remove all surveillance risk.

## Safe Defaults in This Project

- Local-only operation after install.
- No telemetry or tracking.
- Encrypted local state for profiles/templates.
- Minimal bounded logs.

## User Responsibilities

- Verify files before install/import/update.
- Keep trusted signer keys from verified sources.
- Avoid sharing logs containing sensitive network context.
- Use neutral app naming if personal risk is high.

## High-Risk Situations

- If under immediate risk, disconnect quickly.
- If confiscation risk is high, clear app data and remove imported bundles.
- Do not keep unnecessary copies of bundle/update files.

## Social Safety

- Do not pressure others to install without explaining risks.
- Share only verified files; avoid unverified reposts.
