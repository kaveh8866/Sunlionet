# Security Policy

## Supported Versions

Only the following versions of SunLionet receive security updates:

| Version | Supported |
| :--- | :--- |
| 0.3.x-rc1 | Yes |
| < 0.3.0 | No |

## Reporting a Vulnerability

We take the security of SunLionet seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please follow our **[Vulnerability Response Lifecycle](docs/governance/vulnerability-response.md)**.

### Summary of Disclosure Process:
1. **Report**: Email `security@sunlionet.org` with a GPG-encrypted description of the issue.
2. **Triage**: Our Security Response Team (SRT) will acknowledge and triage the report within 24 hours.
3. **Fix**: Valid vulnerabilities will be patched within 24-72 hours for P0/P1 issues.
4. **Disclose**: A Security Advisory will be published after the fix is released.

Thank you for helping keep SunLionet secure!
